Castcading Style Sheets, or better known as CSS, is a language used to style a webpage. It works hand in hand with HTML in order to diplay it's elements on a webpage. CSS can control the layout of multiple webpages at the same time.

CSS saves a load of work by removing style from HTML. Usually, style definitions are stored in external .css files. But with an external stylesheet file, you can change an entire webpage's look with just one file.

In CSS syntax, the selector points at the HTML element you want to style. Semicolons separate declarations in the declaration block. Every declaration will contain a property name and value separated by a colon.
